Passo a passo — Netlify Drop

Opção recomendada — sem cadastro, sem configuração
1
Salve o arquivo do formulárioBaixe o index.html que foi gerado para o seu computador ou celular.
2
Personalize o número do prestadorAbra o arquivo em qualquer editor de texto e troque 5531999999999 pelo seu número real (com DDD, sem espaços ou traços).
3
Acesse app.netlify.com/dropArraste o arquivo index.html para a área de drop da página. O Netlify vai gerar automaticamente uma URL como https://amazing-name-123.netlify.app
4
Gere o QR Code com a URL públicaCopie a URL gerada e cole em qr-code-generator.com para gerar o QR real para impressão.
5
Imprima ou compartilhe!Coloque o QR Code em cartões de visita, panfletos ou stories do Instagram. Quem escanear cai direto no formulário.
💡 Dica: No Netlify, você pode personalizar a URL em Site settings → Change site name para algo como joao-eletricista.netlify.app

O que personalizar no HTML

Troque esses dois valores antes de fazer o deploy
// Linha ~180 do index.html — WhatsApp do prestador:
const PRESTADOR_WA = '5531999999999'; ← seu número aqui

// Linha ~185 — e-mail do prestador (opcional):
const PRESTADOR_EMAIL = 'prestador@exemplo.com.br'; ← seu e-mail

// Título do formulário (linha ~10 aprox):
<title>Solicite seu Orçamento</title>

// Nome exibido na hero (linha ~70 aprox):
<h1>Solicite seu Orçamento</h1>
